Aarambha Real Estate is a lightweight and highly-customizable theme to provide multiple listing services of real estate businesses to make an awesome online presence.

== Description ==

Aarambha Real Estate is one of the generic themes with excellent multipurpose features that can be easily customized for showcasing properties of Marketplace, Real Estate Agents/Agencies, corporate, and real estate websites. The theme brings drag and drop header and footer builder to make it easier for user's to build any type of header and footer in minutes. Moreover, It provides various design variations and templates for displaying properties and services online. With the integration of a powerful crucial real estate plugin, the theme facilitates agent/agency support. This helps to create new agents and allows them to add property listing to your website.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

Aarambha Real Estate includes support for Crucial Real Estate and for Infinite Scroll in Jetpack.
